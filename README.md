# Web: Hugo

### Create
    hugo new site <name>                            --> Creates a new site project
    hugo                                            --> Build site
    hugo server                                     --> Starts the Hugo server at Port 1313

### Themes
    git submodule add <link>.git themes/<loc>       --> Adds a theme submodule
    Explore Themes                                  --> https://themes.gohugo.io/
